
Before starting the script :- 
In the docker_registry_image_addition.pl, replace <path_to_the_version_file> with the 
actual path of the versions.yaml file.

docker_registry_image_addition.pl reads the version.yaml provided with the script.
version.yaml contains the complete list of the images refered by the treasuremap.

Usage
$ perl docker_registry_image_addition.pl
